# FreeCode
AI-Powered Coding Assistant

FreeCode is a simple, open-source AI coding assistant that helps developers write code more efficiently. It provides intelligent code suggestions, explanations, and basic code generation capabilities.

## Features

- Code generation from natural language descriptions
- Code explanation and documentation
- Simple command-line interface
- Extensible architecture for additional AI providers

## Installation

1. Clone this repository:
```bash
git clone https://github.com/teem02/FreeCode.git
cd FreeCode
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the AI coding assistant:
```bash
python freecode.py
```

Follow the interactive prompts to get AI-powered coding assistance.

## Examples

- Generate a Python function: "Create a function to calculate fibonacci numbers"
- Explain code: "Explain what this code does: [paste your code]"
- Get coding help: "How do I sort a list in Python?"
